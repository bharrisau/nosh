# Plan 02-01: Auth primitives and the ssh-agent signing round-trip

## Objective
Add the `nosh-auth` building blocks: Ed25519 SSH key loading (identity, host
key, authorized_keys, known_hosts), SPKI construction, the self-signed-cert
minting via a custom rcgen `SigningKey`, and the rustls `SigningKey`/`Signer`
that routes the CertificateVerify signature to ssh-agent — plus the EARLY
focused test proving the agent Ed25519 round-trip verifies. Validates AUTH-04's
signing path and AUTH-03's key material before verifiers/wiring depend on it.

## must_haves
- truths:
  - "Ed25519-only: non-Ed25519 keys are rejected at load with a clear message (D-12)"
  - "The private key is never read by nosh — signing goes through ssh-agent (AUTH-04, D-04)"
  - "A self-signed cert whose SPKI is the SSH Ed25519 public key can be minted (D-09)"
  - "The ssh-agent Ed25519 signature over a known input verifies with ed25519-dalek"

<task id="3" type="execute">
<action>
Create crates/nosh-auth/src/signer.rs. Define trait-object `RawEd25519Signer: Send+Sync`
with `fn sign(&self, msg: &[u8]) -> anyhow::Result<[u8;64]>` and `fn public_key32(&self) -> [u8;32]`.
Implement `AgentSigner` (holds `SSH_AUTH_SOCK` path + `ssh_key::PublicKey`) whose `sign`
connects via `ssh_agent_client_rs::Client::connect` and calls `sign(pubkey, msg)`, returning
the raw 64-byte sig. Implement `InProcessEd25519Signer` (holds `ed25519_dalek::SigningKey`)
for the fallback/tests. Implement `MintingKey(Arc<dyn RawEd25519Signer>)` as a `rcgen::SigningKey`
(`der_bytes` = pubkey32, `algorithm` = `&rcgen::PKCS_ED25519`, `sign` = delegate) and a
`mint_self_signed_cert(&Arc<dyn RawEd25519Signer>) -> Result<CertificateDer<'static>>`.
Implement `AgentSigningKey` as a `rustls::sign::SigningKey` whose `choose_scheme` returns an
`AgentRustlsSigner` for `SignatureScheme::ED25519` (delegating `sign` to the inner signer,
wrapped so the blocking agent call is acceptable on the handshake task) and whose `public_key`
returns the Ed25519 SPKI.
