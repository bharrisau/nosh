<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| terminal/PTY → client predictor input | Local keystrokes (already trusted; routed unchanged to server). Predictor only reads a copy to speculate display. |
| server StateDiff datagram → predictor cull | Server-confirmed grid is the authority; predictions are reconciled against it. |
| predictor → local stdout (via screen render) | Predicted cells are display-only and must never leak unechoed (password) characters. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-01 | Information disclosure | predictor noecho suppression (`read -s` / `stty -echo`) | mitigate | confirmed_epoch never advances without a server echo → predictions stay tentative → `cell_at` returns None; structural in `cull` + `is_tentative`; proven by a unit test in Task 2 and adversarially in Plan 03 (D-15-04 / PREDICT-04) |
| T-15-02 | Tampering | prediction must never write to the wire | mitigate | `on_input` mutates only local overlay state; keystrokes are forwarded by the unchanged `send_input` path in the integration plan. This module has no `SendStream`/network handle — structurally cannot write to the wire. Verified: predictor.rs imports no quinn send types. |
| T-15-03 | Denial of service | unbounded `pending` VecDeque growth (paste/bulk flooding) | mitigate | Bulk input (>BULK_SUPPRESS_THRESHOLD bytes) and bracketed paste suppress enqueue and reset the epoch; cull removes confirmed/expired predictions every datagram; reset() clears all on mismatch — pending cannot grow without bound under adversarial input. |
| T-15-04 | Tampering | CJK width miscount corrupting predicted columns | mitigate | unicode-width `width()` (not `width_cjk`); ambiguous/combining/ZWJ → epoch reset (D-15-03); wide char at right edge → become_tentative (Pitfall 6). |
| T-15-SC | Tampering | npm/pip/cargo installs (`unicode-width`) | mitigate | RESEARCH.md Package Legitimacy Audit: `unicode-width` 0.2.2 — crates.io, ~10 yrs, rust-lang org (unicode-rs), slopcheck [OK], Approved. No [ASSUMED]/[SUS] packages → no human checkpoint required. |
</threat_model>
